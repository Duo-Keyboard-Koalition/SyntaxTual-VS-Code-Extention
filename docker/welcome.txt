
============================================================
  Claude Code + VS Code (browser)
============================================================
  You are the non-root user 'user' (sudo/admin, no password).

  VS Code is at http://localhost:8080

  FIRST-RUN SETUP — authenticate Claude with your account:

      claude

  Follow the login link it prints to sign in to your Anthropic
  account. Done once; saved in ~/.claude and persisted.

  Danger mode (skips permission prompts):

      claude-danger        # = claude --dangerously-skip-permissions

  Works because this container runs as non-root 'user'.
